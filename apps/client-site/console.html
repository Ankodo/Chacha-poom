Setup Instructions
  1. Install v2rayNG or NekoBox from Google Play Store or GitHub releases.
  2. Open the app, tap the + button, then select "Import from clipboard" or "Add subscription".
  3. Paste your subscription link: copy it using the button above.
  4. Tap Update subscription to fetch server configs. All protocols and servers will appear automatically.
  5. Select a server, tap the Connect button. Allow VPN permission when prompted.

  5. Select a server and tap Connect. Allow VPN configuration when prompted by iOS.
